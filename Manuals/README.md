## Key Reference Info

### Gecko G540

- Input voltage: 18–50V DC
- Max current per axis: 3.5A
- Microstepping: fixed 10x
- Charge pump: required — disables drives if Mach3 not running
- Current set resistor formula: `R = 47kΩ × (I / 3.5)`
- Fault indicator: red LED on G540 front panel
