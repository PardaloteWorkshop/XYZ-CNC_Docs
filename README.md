# XYZ-CNC Documentation

> Original XYZ-CNC kit homepage (archived): https://web.archive.org/web/20220307203536/https://xyz-cnc.com/

Australian-made CNC router kit, now discontinued. This repo documents the machine as-built, including wiring, settings, modifications, and reference material for ongoing operation and maintenance.

---

![CNC](../Images/XYZ-CNCimage01.jpg)

## Machine Overview

| Spec | Detail |
|------|--------|
| Machine | XYZ-CNC Router Kit |
| Origin | Melbourne, Australia |
| Acquired | March 2025 |
| Age at acquisition | ~12 years |
| Controller | Gecko G540 |
| Control Software | Mach3 |
| Spindle | 80mm water-cooled |
| Spindle power | 2.2kW |
| VFD | Huanyang HY02D223B |
| Bed size (X × Y) | 2640mm × 1250mm |
| Z travel | 150mm |
| Frame | 100x50x3 Steel Tubing |
| Drive | Leadscrew / Roller Chain |
| Steppers | NEMA23 |
| OS / PC | Windows 7 32bit / Dell Optiplex 780 |
